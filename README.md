# Sanket Shigaonkar — Portfolio

A minimal, dark-themed developer portfolio built with Next.js 16 and Tailwind CSS 4.

**Live:** [sanketshigaonkar.vercel.app](https://sanketshigaonkar.vercel.app)

## Features

- **Responsive Design** — Optimized for desktop and mobile
- **Smooth Animations** — Mouse spotlight effect, hover transitions
- **Sticky Navigation** — Section-aware nav with scroll tracking
- **Accessible** — Semantic HTML, ARIA labels, keyboard navigable
- **Fast** — Minimal dependencies, optimized fonts, Vercel Analytics

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Fonts:** Inter (Google Fonts)
- **Analytics:** Vercel Analytics
- **Deployment:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
